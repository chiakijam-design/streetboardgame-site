# 個人情報漏えい・漏えい疑い対応手順

最終更新: 2026-07-22

## 1. 連絡体制

- 対応責任者: Streetboardgame運営者 平川智章
- 緊急連絡: 090-4707-5225
- 利用者窓口: `https://www.streetboardgame.com/contact`
- 技術確認先: Cloudflare Workers / D1 / R2、Stripe、Google / YouTube、Formspree、GA4の各管理画面とサポート

発見者は、個人データが実際に流出したと断定できない段階でも、アクセス権限の誤設定、不審な管理ログ、秘密URL・APIキーの公開、第三者からの指摘を「漏えい疑い」として直ちに対応責任者へ連絡する。

## 2. 発見から最初の60分

1. 発見時刻、発見者、対象システム、画面・ログ・URL、推定件数を記録する。
2. 証拠となるログや設定画面を保存する。調査前にログやD1行を削除しない。
3. 漏えい経路を封じる。
   - 管理画面: `LIVE_ADMIN_TOKEN`、`LIVE_ADMIN_TOTP_SECRET`、`LIVE_ADMIN_SESSION_SECRET`を交換
   - スタッフ・本人URL: 運営コンソールから再発行
   - API・Webhook: 該当secretを交換し、不要なトークンを失効
   - 公開R2・D1設定: 公開アクセスを停止し、最小権限へ戻す
4. 影響が継続する場合はLIVEを`maintenance`へ切り替え、新規作成・参加を停止する。
5. Cloudflare、Stripe、Google、Formspree等の委託先が関係する場合は、各社のインシデント窓口へ連絡する。

## 3. 24時間以内の影響評価

次を一覧化する。

- 発生期間と原因
- 対象データの項目。氏名、メール、IP、決済ID、チャンネル情報、画像、投票等
- 対象人数と対象ルーム・購入ID
- 閲覧、取得、改ざん、消失の別
- 暗号化・ハッシュ化の有無
- 二次被害。不正決済、なりすまし、秘密URL悪用等
- 現在もアクセス可能か

要配慮個人情報、財産的被害のおそれ、不正目的による漏えい、または1,000人を超える漏えい等に該当する可能性がある場合は、個人情報保護委員会への報告対象として扱う。判断が難しい場合は専門家または個人情報保護委員会の相談窓口へ確認する。

## 4. 報告・本人通知

- 報告対象の場合、個人情報保護委員会へ概ね3〜5日以内に速報する。
- 確報は原則30日以内。不正目的によるおそれがある場合は60日以内に行う。
- 対象者には、状況に応じて速やかに、概要、対象情報、原因、二次被害の可能性、利用者が行う対策、問い合わせ先を通知する。
- 個別通知が困難な場合は、Web告知と専用問い合わせ窓口を用意する。

通知文には、未確認事項を断定せず、「確認済み」「調査中」「影響なしを確認」を区別して記載する。

## 5. 復旧条件

1. 漏えい経路が閉じ、交換前のsecret・URL・セッションが使えない。
2. D1・R2・Stripe・Cloudflare・Google・Formspree・GA4のアクセス権限を再点検した。
3. 不正アクセスとデータ改ざんの範囲を確認した。
4. 必要な報告・本人通知を開始した。
5. 再発防止策をテストし、対応責任者がサービス再開を承認した。

## 6. 事後対応

- 発生、検知、封じ込め、報告、復旧の各時刻を記録する。
- 原因、影響、判断根拠、利用者対応、再発防止策をまとめる。
- 保存期間を超えて証拠を保全する場合は、対象、理由、保全期限、閲覧権限を記録する。
- 30日以内に手順書、監視、権限、委託先設定を見直す。
