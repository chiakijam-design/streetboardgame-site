1. 事業者

Streetboardgame運営者　平川智章（以下「運営者」）

2. 取得する情報

  • 通常ゲーム：入力した名前、回答、結果、端末内に保存される設定
  • LIVE：ゲームタイトル、YouTubeチャンネルURL・チャンネルID・チャンネル名、公開動画のタイトル等、問題・選択肢、予約日時、参加者名、投票・回答、ルーム操作履歴
  • YouTuber登録：氏名・事業者情報、連絡先、チャンネル情報、登録画像、Stripe ConnectアカウントID、収益・返金・支払記録
  • 契約同意：規約バージョン、同意日時、IPアドレス、ConnectアカウントID、利用端末・ブラウザ情報
  • 購入：商品、価格、購入日時、Stripeの決済・顧客・領収書識別子、メールアドレス、ダウンロード状況。カード番号は運営者のサーバーに保存しません。
  • 問い合わせ・アクセス：氏名、メールアドレス、問い合わせ内容、IPアドレス、Cookie、閲覧ページ、参照元、端末情報、障害ログ

3. 利用目的

  • ゲームの作成、進行、投票集計、結果表示および再開
  • YouTubeチャンネルに関連する問題候補の生成
  • 本人確認、契約管理、決済、返金、収益計算、不正利用・チャージバック対策
  • 有料結果画像の生成、配信、30日間の再ダウンロード提供
  • 問い合わせ対応、障害対応、品質・セキュリティ改善
  • 法令、会計・税務上必要な記録の保存

4. YouTube API Services

本サービスはYouTube API Servicesを利用する場合があります。本サービスはYouTube、Googleまたはその関係会社が提供、後援、承認するサービスではありません。

公開チャンネル・動画の情報を取得して問題候補の作成に利用します。YouTubeのログイン認証情報やパスワードを収集・保存しません。YouTube機能の利用にはYouTube利用規約およびGoogleプライバシーポリシーが適用されます。

将来Googleアカウント認証を導入した場合、利用者はGoogleセキュリティ設定で連携を解除できます。削除依頼を受けた対象データは、法令上保存が必要なものを除き削除します。

5. 外部サービス・委託先

Cloudflareサイト配信、データ保存、セキュリティおよびアクセスログ
Stripe決済、本人確認、Connectアカウント、返金・収益分配
Google / YouTube公開チャンネル・動画情報の取得、アクセス解析
Formspreeお問い合わせフォームの受信

これらの事業者が国外でデータを取り扱う場合があります。運営者は、契約・各社の保護措置・適用法令を確認し、必要な安全管理措置を講じます。

6. 第三者提供

本人の同意がある場合、業務委託に必要な場合、事業承継の場合、または法令に基づく場合を除き、個人データを第三者に提供しません。YouTuberには、個々の購入者を特定できる情報ではなく、原則として売上・返金・分配に必要な集計情報を提供します。

7. 保存期間

  • LIVEの企画・投票データ：予約枠終了または利用終了後、運用上必要な期間を経て順次削除
  • 有料画像：購入日から30日間ダウンロード可能。その後は順次削除
  • 契約、決済、返金、分配、会計・税務記録：契約期間中および法令上必要な期間
  • セキュリティログ：不正対策および障害調査に必要な期間
  • 問い合わせ：対応完了後、紛争対応・再照会に必要な期間

具体的な保存期間は、法令、紛争・不正調査、バックアップの都合により延長される場合があります。

8. Cookie・アクセス解析

本サービスは、機能維持、利用状況の把握および改善のためCookie、localStorageその他の類似技術を利用します。ブラウザ設定でCookieを無効化できますが、一部機能が利用できなくなる場合があります。

9. 安全管理

アクセス制御、通信の暗号化、権限分離、ログ監視、委託先の選定など、取り扱う情報の性質に応じた安全管理措置を講じます。秘密URLや参加コードは、利用者側でも第三者に漏れないよう管理してください。

10. 開示・訂正・削除等

保有個人データの開示、訂正、利用停止、削除その他法令上の請求は、お問い合わせフォームからご連絡ください。本人確認後、法令に従って対応します。ゲーム進行中の投票データなど、削除すると他の参加者の利用に影響する情報は、終了後の削除となる場合があります。

未成年者は、未成年者利用規定に従い、必要に応じて親権者等の法定代理人の同意を得て利用してください。
