4. YouTube API Services

本サービスはYouTube API Servicesを利用する場合があります。本サービスはYouTube、Googleまたはその関係会社が提供、後援、承認するサービスではありません。

公開チャンネル・動画の情報を取得して問題候補の作成に利用します。YouTuber本人がGoogle認証を選んだ場合は、本人が管理する過去1年の公開動画から、再生数を主軸に公開日の新しさも加味して優先した最大8本の字幕を取得し、実際の発言・場面を反映した問題候補の生成に利用します。Googleアクセストークンは処理後に失効させ、ログイン認証情報、パスワード、アクセストークンおよびリフレッシュトークンを保存しません。YouTube機能の利用にはYouTube利用規約およびGoogleプライバシーポリシーが適用されます。

将来Googleアカウント認証を導入した場合、利用者はGoogleセキュリティ設定で連携を解除できます。削除依頼を受けた対象データは、法令上保存が必要なものを除き削除します。

8. Cookie・アクセス解析

本サービスは、機能維持、利用状況の把握および改善のためCookie、localStorageその他の類似技術を利用します。ブラウザ設定でCookieを無効化できますが、一部機能が利用できなくなる場合があります。

9. 安全管理

管理画面の二要素認証と短期セッション、ゲーム用D1と購入履歴用D1の分離、R2の非公開化、Content Security Policy、通信の暗号化、権限分離、期限切れデータの自動削除、ログ監視を実施します。秘密URLや参加コードは、利用者側でも第三者に漏れないよう管理してください。

個人データの漏えいまたはそのおそれを把握した場合は、アクセス停止、証拠保全、影響範囲確認、委託先への連絡を行い、法令上必要な場合は個人情報保護委員会への報告と対象者への通知を行います。

10. 開示・訂正・削除等
